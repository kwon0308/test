# markdown이란?
markdown은 텍스트를 서식있는 문서로 쉽게 변환 할 수 있는 마크업 언어

# 마크업 언어란?
텍스트에 태그를 사용하여 문서의 구조와 서식을 정의하는 언어
예) HTML, XML..

1. **헤더(heades)**:
```
  백틱(`) <-- 3개 코드라인 markdown 문법이 적용 안됨.
  # 헤더1
  #헤더2
  #헤더3
```
##헤더2

2. 굵게
